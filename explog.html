Exponents and Logarithms¶

derivater.exp(exponent)[source]¶

This is equivalent to e**exponent.

Unlike in Python’s math module, this function is no more precise than a plain e**x, and it’s there just for people who prefer to write exp(x) instead of e**x.

>>> exp(x)
e**x
>>> exp(x).derivative(x)
e**x
derivater.ln(numerus)[source]¶

Return the natural logarithm (base e) of numerus.

>>> ln(e)
1
>>> ln(x).derivative(x)
1 / x
>>> (x**x).derivative(x)
x**x*(ln(x) + 1)
derivater.log(numerus, base=e)[source]¶

Return the logarithm of numerus with the given base.

This is equivalent to ln(numerus) / ln(base).

>>> log(x, 2)
ln(x) / ln(2)
>>> log(x)
ln(x)
derivater.log2(numerus)¶

This is a convenience function that returns log(numerus, 2).

derivater.log10(numerus)¶

This is a convenience function that returns log(numerus, 10).

class derivater.NaturalLog(numerus)[source]¶

The type of many objects returned by ln().

Don’t create NaturalLog objects yourself; use ln() instead. It may return special values like 0 or 1 instead of NaturalLog objects.

numerus¶

The object passed to ln().
