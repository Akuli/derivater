class derivater.sin(arg)[source]¶

The sine function.

>>> sin(2*x + 1)
sin(2*x + 1)
>>> sin(2*x + 1).derivative(x)
2*cos(2*x + 1)
class derivater.cos(arg)[source]¶

The complement sine function, mathematically equal to sin(tau/4 - arg).

>>> cos(x).derivative(x)
-sin(x)
