Constants¶

Derivater comes with a bunch of handy constants.

derivater.e¶

Euler’s number, approximately 2.718281828459045.

derivater.tau¶

A full turn in radians, approximately 6.283185307179586.

derivater.pi¶

A convenient way to write tau/2.

All of the above constants are actually just instances of this class:

class derivater.NamedConstant(name)[source]¶

Represents a constant with a name.

The biggest difference between this class and Symbol is that two NamedConstants with the same name don’t compare equal:

>>> NamedConstant('e') == e
False

This means that there’s exactly one Euler’s number, documented as e above, and if you make another constant called e it won’t be mistakenly treated as Euler’s number.

>>> ln(e)
1
>>> ln(NamedConstant('e'))
ln(e)

If you want to use e.g. a Greek letter for the name, you can do e.g. goldenratio = NamedConstant('\N{greek small letter phi}') to get a φ constant.
